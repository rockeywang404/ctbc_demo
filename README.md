# 中信託管與支付服務 demo
## 專案簡介
這是一個前端介面的Prototype，主要用於呈現企業資金入金、RWA 理財、金庫託管以及結匯出金等核心流程的視覺畫面。目前為靜態展示版本，僅供團隊內部確認介面流程與排版使用，尚未串接真實的後端系統與數據。

### 如何測試與瀏覽
為了方便大家快速檢視，這個 Demo 不需要安裝任何程式碼編輯器或伺服器軟體。請直接依照以下步驟，即可在自己電腦上預覽畫面：

#### 步驟一：下載與準備檔案
請將收到的檔案（或壓縮檔）下載並儲存到您的電腦桌面或指定資料夾。

**重要提醒：** 如果您下載的是壓縮檔（.zip），請務必先對檔案點擊滑鼠右鍵，選擇「解壓縮全部」或「解開壓縮檔」。請不要在未解壓縮的狀態下直接預覽，這可能會導致網頁排版破裂或圖片無法顯示。

#### 步驟二：開啟網頁檔案
進入解壓縮後的資料夾。

找到副檔名為 .html 的網頁檔案（檔名為 中信畫面v2.0.html）。這個檔案的圖示通常會是您電腦預設瀏覽器的圖示，例如 Google Chrome、Edge 或 Safari。

直接對著該檔案連續點擊兩下（雙擊）。

#### 步驟三：開始測試
檔案會自動在您的網頁瀏覽器中打開，您現在可以直接在畫面上進行點擊測試，檢視各個步驟的排版與切換效果。

**進階開啟方式（若雙擊無效）：**
如果您連續點擊兩下沒有反應，或是打開了不正確的程式，請對著該 .html 檔案點擊滑鼠右鍵，選擇「開啟檔案」（或 Open With），然後從選單中點選 Google Chrome（建議使用 Chrome 以獲得最穩定的瀏覽體驗）。

### 常見問題排解
Q: 為什麼網頁看起來亂糟糟的，或是沒有顏色？
A: 請確認您是否有確實完成「解壓縮」的動作。

Q: 我點擊了網頁上的「送出」或「申購」按鈕，為什麼沒有反應？
A: 目前這份檔案為視覺與流程的展示用Demo，部分按鈕僅具備視覺效果，尚未有真實的資料庫運算功能。
